# 樂透刮刮樂吉祥卡 (Lottery Scratch Card)

這是一個專為馬年設計的數位刮刮樂網頁應用程式，結合了 401 班級的企鵝特色元素。使用者可以透過互動式刮除體驗，獲得隨機的馬年吉祥話。

## 🚀 專案特點
- **馬年主題**：結合傳統春節氛圍與馬年吉祥語。
- **401 班級特色**：視覺設計融入可愛的企鵝元素。
- **行動裝置優化**：針對 Android/iOS 的觸控體驗優化，特別適配 LineBot WebView。
- **中英雙語**：支援繁體中文與英文切換。
- **RWD 設計**：自動適配各種螢幕尺寸。
